<!-- STORY -->

### Usage

```html
<div class="card px-2">
  <div class="e-navline">
    <ul class="nav">
      <li class="nav-item"><a href="" class="active nav-link">Home</a></li>
      <li class="nav-item"><a href="" class="nav-link">Bookmarks</a></li>
      <li class="nav-item"><a href="" class="nav-link">Messages</a></li>
    </ul>
  </div>
</div>
```
